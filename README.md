# Namaskara AI — Official Website

Premium website for Namaskara AI, Assam's first native AI platform.

## Tech Stack
- React + Vite
- Three.js / React Three Fiber (3D Xorai orb)
- GSAP + ScrollTrigger (scroll animations)
- Lenis (smooth scroll)
- Plus Jakarta Sans / Inter typography

## Design
- Cyberpunk × Assamese cultural aesthetic
- Colors: Gamosa Red `#8B1A1A`, Ivory `#F5EDD6`, Tea Garden Green `#2D5A27`, Gold `#C9962A`
- Custom cursor, loading screen, scroll progress indicator
- 3D rotating Xorai-inspired orb with particle field

## Development

```bash
npm install
npm run dev
```

## Deploy
Push to `main` branch — GitHub Actions auto-deploys to GitHub Pages.

Live at: `https://<username>.github.io/namaskara-ai/`
